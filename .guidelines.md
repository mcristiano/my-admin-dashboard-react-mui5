# Guidelines

## JSX Conditional Rendering

Remember that in JSX, we use `&&` for conditional rendering, **not** `&amp;&amp;`.

**Correct:**

```jsx
{condition && <Component />}
```

**Incorrect:**

```jsx
{condition &amp;&amp; <Component />}
```
